# MONCASA
# "몬까사, 포근함을 완성하다"
Bootstrap 기반 반응형 광고 이벤트 페이지를 통해, 고객에게 친환경(ESG) 가치를 선보입니다.
2050세대의 라이프스타일을 공략하는 맞춤형 가구/인테리어 경험을 제공하는 글로벌향 프로젝트입니다.
